# Cinquième Pouvoir Associatif — mini site statique

## Pages
- `index.html` : accueil (navigation)
- `dossier-blanc.html` : le document principal
- `page-ressources.html` : page suivante (structure à compléter)
- `mentions.html` : mentions / cadre d’usage
- `assets/site.css` + `assets/site.js` : style et fonctions partagées

## Publier sur GitHub Pages (rapide)
1. Créer un dépôt GitHub (public).
2. Mette ces fichiers à la racine du dépôt.
3. Aller dans **Settings → Pages**.
4. Source : **Deploy from a branch**, branche **main**, dossier **/** (root).
5. Sauvegarder : le site sera accessible via l’URL GitHub Pages.

## Modifier le contenu
- Ouvrir `page-ressources.html` et remplacer les blocs “À compléter”.
- Ajouter des fichiers (PDF, images) dans un dossier `docs/` ou `assets/` puis faire des liens.
