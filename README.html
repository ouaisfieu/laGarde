# Cinquième Pouvoir Associatif — mini site statique ## Pages - `index.html` : accueil (navigation) - `dossier-blanc.html` : le document principal - `page-ressources.html` : page suivante (structure à compléter) - `mentions.html` : mentions / cadre d’usage - `assets/site.css` + `assets/site.js` : style et fonctions partagées ## Publier sur GitHub Pages (rapide) 1. Crée un dépôt GitHub (public). 2. Mets ces fichiers à la racine du dépôt. 3. Va dans **Settings → Pages**. 4. Source : **Deploy from a branch**, branche **main**, dossier **/** (root). 5. Sauvegarde : ton site sera accessible via l’URL GitHub Pages. ## Modifier le contenu - Ouvre `page-ressources.html` et remplace les blocs “À compléter”. - Ajoute des fichiers (PDF, images) dans un dossier `docs/` ou `assets/` puis fais des liens.
